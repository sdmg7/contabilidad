Menu with different HTML elements

  • This demo uses theme winxp1.
  • This menu shows the different HTML elements that you can include in your menu.
  • Not only can you use different fonts sizes and effects, but you can also include form elements like checkboxes, radio buttons, and text input boxes.
  • Notice the scrolling list under forms. It allows you to create a long list of items in a menu that the user can easily scroll through.
