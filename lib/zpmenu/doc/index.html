The Zapatec DHTML Menu Overview    
A navigation bar or menu is one of the most important elements of your website. The Zapatec DHTML Menu is a search engine friendly, attractive, and versatile way to help your users navigate your web site. Use the Zapatec DHTML Menu as a site map, an application menu or a way to display your data. While the Zapatec DHTML Menu works in all modern browsers, it degrades gracefully in older browsers -- it displays the underlying HTML code.

Start by checking out our demo menus and see the various ways the Zapatec DHTML Menu can be used.
The feature list describes the different features that make the Zapatec DHTML Menu unique.
The Menu Wizard lets you try different aspects of the Menu. You can choose a theme, make the menu vertical or horizontal, edit the menu items, without writing a single line of javascript.
