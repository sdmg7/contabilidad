Different Themes in the Menu
Shows a menu where you can try the different themes that come with the menu.
HTML Elements in the menu
Shows how you can include various HTML items in your menu including form elements.
Vertical Menu
Shows a vertical rather than a horizontal menu.
Click Menu
Menu where you need to click, rather than hover, for the menu items to drop down.
Two Menus
You can have multiple menus on one page. This example shows two menus. Also, the first menu will keep track of the user menu item clicks, see rememberPath option.
Control Open and Close
Set the interval at which the menus opens and close
Zapatec Website Menu
A menu for the Zapatec web site, where clicking on the menu items will take you to the corresponding page.
Menus with Glide and Fade Effects
A menu demonstrating the glide effect and the menu demonstrating the fade effect.
Menu with Tooltips and Clickable Headers
The Zapatec site menu with tooltips for the different menu items. The top menus are clickable as well as the menu items.
Menu with Icons
This menu shows a sample of the icons included as part of the Zapatec DHTML Menu including icons such as buy, shopping cart, products, services, home, etc.
Different Styles for Different Menu Items
Customize styles for specific sub-menus or menu items.
Draggable menu
Drag this menu to a different part of the page.
Scrolling Menu
This menu scrolls/floats on the page as the user scrolls down the page.
Drop Shadow
This menu demos the Drop Shadow feature.
Context Sensitive Menu - Many Themes
Context Sensitive Menu using many themes. A nice demo to show how Context Sensitive Menus and Dynamic Themes can give a custom look and feel to your website.
Target Property
Demo how a menu links can use the target property. Demo shows how you can have a custom target, _blank, _new, _top.
Disable Menu Items
Demo how you can dynamically disable/enable menu items. A good method to disable a link for the current page in a navigation menu.
Long Scrolling Menu
See how you can scroll up or scroll down for a long sub-menu, one that does not fit vertically on the screen.
Absolute Position
See how you can easily absolutely position the menu.
Track Previous Menu Location
Track the location of previous menu clicks. Very useful for navigation type menus.
Stretch
Menu is stretched to the container size. Useful to have the menu cover the full width of container.
